humanoid
========

Humanoid in Python

Programs needed to run this code:
Python 2.7+
GLUT

Type python humanoid.py in console to run it.

Controls:

D/d - torso
E/e - head 1
R/r - head 2
S/s - left upper arm
A/a - left lower arm
F/f - right upper arm
G/g - right lower arm
X/x - left upper leg
Z/z - left lower leg
C/c - right upper leg
V/v - right lower leg
Q/q - exit
